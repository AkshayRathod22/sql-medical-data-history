# Medical Data History – SQL Project

This project was completed as part of an internship program and focuses on analyzing medical data using SQL queries.

## Project Objective
To retrieve, analyze, and summarize medical patient data using SQL in order to answer business and analytical questions.

## Project Description
The project involves writing multiple SQL queries to work with medical datasets such as patients, admissions, and related attributes. 
All tasks are query-based and emphasize SQL logic, filtering, aggregation, and date handling.

## Key SQL Concepts Used
- SELECT, WHERE
- ORDER BY
- GROUP BY, HAVING
- Aggregate functions (COUNT, SUM)
- Date functions
- Aliasing
- Sorting and filtering data

## Tasks Performed
- Retrieve patient demographic details
- Filter patient records based on conditions
- Analyze admissions by date
- Group patients by weight and height ranges
- Count records using aggregation
- Sort and format query results

## Tools Used
- SQL
- MySQL (or mention your DB if different)

## Project Files
- `medical_queries.sql` – Contains all SQL queries written for the project

## Note
This is a query-based SQL project. No dashboards or visualizations were created as the focus was on SQL data analysis and query writing.
